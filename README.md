# 📊 Full-Stack CRM Application

A high-performance Customer Relationship Management (CRM) platform designed to streamline business workflows, track client interactions, and manage data pipelines efficiently. Built with the **PERN stack** (PostgreSQL, Express, React, Node.js).

🚀 **[View Live Demo](https://crm-frontend-seven-rho.vercel.app/)**

---

## ⚡ Key Highlights
* **Full-Stack Architecture:** Completely decoupled frontend (React) and backend (Node/Express) for optimal scalability.
* **Relational Database Management:** Structured PostgreSQL schema designed for robust data integrity and fast querying.
* **Modern UI/UX:** Clean, intuitive interface built with React to deliver a smooth user experience.
* **RESTful API:** Optimized backend routing handling complete CRUD operations for managing leads, clients, and metrics.

---

## 🛠️ Tech Stack

| Layer | Technology |
| :--- | :--- |
| **Frontend** | React.js, HTML5, CSS3, JavaScript (ES6+) |
| **Backend** | Node.js, Express.js |
| **Database** | PostgreSQL |
| **Deployment** | Vercel (Frontend) |
